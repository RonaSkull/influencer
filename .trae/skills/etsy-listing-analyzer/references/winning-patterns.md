# Winning Etsy Listing Patterns

## Image Patterns That Convert

### Thumbnail (Main Image) Requirements
1. **Readable at 300x300px** - Test by shrinking image
2. **High contrast** - Pops against Etsy's white/gray
3. **Single focal point** - One clear subject
4. **Text minimal or none** - Hard to read at thumbnail size
5. **Styled mockup > flat file** - Frame mockups outperform

### Gallery Best Practices (10 images max)
| Slot | What to Show |
|------|--------------|
| 1 | Hero mockup (best angle, styled) |
| 2 | What's included (all items laid out) |
| 3 | Size comparison or dimensions |
| 4 | Lifestyle shot (in real room) |
| 5 | Close-up detail |
| 6 | Alternate colorway/version |
| 7 | Print instructions or how-to |
| 8 | Bundle contents breakdown |
| 9 | Size variations shown |
| 10 | Shop branding/thank you |

### Mockup Styles That Sell
- **Frame mockups** - White, black, or wood frames on neutral walls
- **Lifestyle** - Actual rooms with decor context
- **Flat lay** - Clean, styled with props
- **Device mockups** - For digital planners/templates
- **Hands holding** - Shows scale, adds human element

### Image Mistakes to Avoid
- ❌ Watermarks (looks amateur)
- ❌ Text-heavy thumbnails (unreadable)
- ❌ Low resolution (looks cheap)
- ❌ Busy backgrounds (distracting)
- ❌ Inconsistent style across gallery
- ❌ Missing size/scale reference

## Title Patterns

### Winning Formula
```
[Primary Keyword] | [Secondary Keyword] | [Audience/Use] | [Format] | Instant Download
```

### Character Limits
- Etsy allows 140 characters
- First 40-50 chars most important (truncated in search)
- Front-load primary keyword

### Examples by Category

**Wall Art:**
```
LDS Temple Print | Salt Lake Temple Wall Art | Modern Latter Day Saint Home Decor | Digital Download
```

**Activity/Bundle:**
```
LDS Easter Bundle | Holy Week Scripture Cards | Primary Activity Printable | Kids Coloring Pages | PDF
```

**Quote Print:**
```
Think Celestial Print | Russell M Nelson Quote | LDS Home Decor | General Conference Wall Art | Download
```

### Separators
- Pipe `|` - Most common, clean
- Dash `-` - Also works
- Comma `,` - Less visual separation
- Bullet `•` - Stylish alternative

## Description Patterns

### Opening Hook (First 160 chars)
This shows in search preview - make it count!

**Winning openers:**
- "Bring [benefit] to your home with this beautiful..."
- "The perfect [occasion] gift for..."
- "Instantly download and print this stunning..."
- "Celebrate [event] with this [product]..."

```markdown

• [item 2]
• [item 3]
```

### Power Words That Convert
- Instant, Immediately, Today
- Beautiful, Stunning, Gorgeous
- Perfect, Ideal, Great for
- Easy, Simple, Quick
- Unique, Special, One-of-a-kind
- Save, Bonus, Included

## Pricing Patterns

### VintageGlow Strategy (Volume Winner)
```
List Price: $3.99
Sale Price: $1.99 (50% off - perpetual)
Volume: Buy 3 = 60% off, Buy 5 = 70% off
```
Result: 1,300+ sales/month

### Premium Strategy (Margin Winner)
```
Singles: $4.99-6.99
Bundles: $12.99-19.99
Ultimate: $24.99-34.99
```
Result: Fewer sales, higher AOV

### Hybrid Strategy (Recommended)
```
Singles: $5.99 → $2.99 (50% off)
Bundles: $19.99 → $9.99 (50% off)
Ultimate: $39.99 → $19.99 (50% off)
Volume discounts on top
```

### Price Anchoring
Always show "original" price crossed out:
- Creates perceived value
- "50% off" triggers urgency
- Etsy shows sale badge in search

## Review Patterns

### What Drives 5-Star Reviews
1. **Files work perfectly** - No tech issues
2. **Looks as pictured** - Accurate mockups
3. **Easy to download** - Clear instructions
4. **Fast response** - Answer questions quickly
5. **Bonus value** - Extra sizes, variations

### Common Complaints (Avoid These)
1. "Colors didn't match" - Add disclaimer
2. "Couldn't figure out download" - Add instructions
3. "Smaller than expected" - Show size clearly
4. "Not what I thought" - Better mockups
5. "No response from seller" - Reply within 24h

## Shop Patterns

### Listing Count Benchmarks
| Listings | Expected Sales |
|----------|----------------|
| 50 | 50-200/month |
| 100 | 150-400/month |
| 250 | 400-1,000/month |
| 500+ | 1,000+/month |

More listings = more lottery tickets

### Category Distribution (VintageGlow Model)
- 40% Seasonal (Christmas, Fall, Easter, etc.)
- 30% Evergreen (quotes, nature, abstract)
- 20% Trending (current aesthetics)
- 10% Experimental (testing new niches)

### Refresh Strategy
- Update seasonal items 4-6 weeks before season
- Renew top performers monthly
- Add "2026" or current year to time-sensitive items
- Archive (don't delete) underperformers
